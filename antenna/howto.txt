Running a simple example:


java -jar antenna.jar <input> <output> <features>
Parameters:
- input: the project you want to preprocess
- output: the result of the preprocessing
- product: FEATURE1 FEATURE2 FEATURE3...

Example with Stack SPL

Stack has 3 features: LOCKING, LOGING and STATISTICS

command line to preprocess Stack with the product: LOCKING STATISTICS
	java -jar antenna.jar ../spl-test-amplification/antenna/StackIFDEF ../spl-test-amplification/antenna/out LOCKING STATISTICS
